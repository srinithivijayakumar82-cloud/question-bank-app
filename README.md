# Question Bank Generator

An AI-powered tool that generates quiz questions from any study content.

## Tech Stack
- Frontend: React + Vite
- Backend: FastAPI (Python)
- AI: Groq API (Llama 3.3 70B)

## Features
- Paste any study content
- Choose difficulty level (easy/medium/hard)
- Choose number of questions
- Get AI-generated questions with answers instantly
